Get Started

  1. Download and Un-zip Forge (or a recent snapshot build ) into a folder on your hard-disk, this folder will be your FORGE_HOME
  2. Add '$FORGE_HOME/bin' to your path ( windows , linux , mac osx )
  3. * Consider installing Git and Maven 3.0.4+ (both optional)
  4. Open a command prompt and run 'forge' (if you are on Windows, you will need to run forge.bat unless using a Unix-style terminal)

What should I do next?

Now you are ready to build an application.
